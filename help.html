Order Management System

The Way Coffe House


Taking a table order:

  1. Take a customers order, as normal, on iZettle.
  2. From the 'Tables' folder add the corresponding table, for that customer, to the order.
  3. Take payment
  4. This order will now be filter out of the takeaway orders and displayed in thetables page. (Click the 'Table orders' button.)

Processing a table order:

#After a few seconds the table order placed will appear in the order feed #The order can be marked as 'in-process' by clicking (or tapping once) on it. It will then turn orange. #Once the tables order is complete, the order can be closed by clicking on it once more, a confirmation window will appear. Upon clicking 'Yes' the order will be closed.
